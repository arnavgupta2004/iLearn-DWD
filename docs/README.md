## Submission Notes

The documentation reflects the final project state:

- single active assessment/submission system using `assessments` and `assessment_submissions`
- AI tutor with RAG over uploaded course materials
- professor analytics and student progress intelligence
- student and professor calendars with interview scheduling
- page-scoped AI assistants
- session-validated student-facing APIs
